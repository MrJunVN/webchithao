# Web Chi Thao

Project moi, tach rieng hoan toan khoi VanThinh va NTS.

## Hien trang

- Da tao frontend React + Vite moi trong `frontend/`
- Da doi giao dien mac dinh thanh scaffold website ban thuc pham bo sung
- Co san cac khu vuc de noi tiep:
  - danh sach san pham
  - combo
  - gio hang demo
  - FAQ
  - khu chatbot

## Chay local

```bash
cd frontend
npm install
npm run dev
```
